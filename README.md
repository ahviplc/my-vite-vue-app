# 创建项目
```markdown
# npm 6.x
npm init @vitejs/app my-vite-vue-app --template vue

# npm 7+, extra double-dash is needed:
npm init @vitejs/app my-vite-vue-app -- --template vue

# yarn
yarn create @vitejs/app my-vite-vue-app --template vue

# 启动
cd my-vite-vue-app

yarn 或者 npm i

npm run dev
```

# 注意
> 使用 npm i 安装 启动报错  
> 后来 又好了
